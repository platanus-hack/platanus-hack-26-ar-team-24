# KnowAhuman

KnowAhuman es una plataforma de matching asistido por inteligencia artificial orientada a conectar talento con startups de una forma más humana, profunda y contextual. En lugar de depender únicamente de filtros estáticos, formularios o coincidencias por palabras clave, el sistema construye agentes representativos de cada usuario y utiliza conversaciones simuladas para explorar afinidad, intereses compartidos, valores, estilo de comunicación y potencial de colaboración.

El proyecto propone una nueva forma de descubrir conexiones relevantes entre personas. Cada usuario puede crear un agente a partir de su perfil personal y profesional, y ese agente actúa como una extensión conversacional capaz de interactuar con otros perfiles. A partir de esas simulaciones, la plataforma genera señales de compatibilidad, resúmenes interpretables y métricas que ayudan a decidir con quién vale la pena avanzar.

## Problema que resuelve

Los procesos de búsqueda de talento, socios o equipos fundadores suelen apoyarse en información incompleta: currículums, bios breves o stacks tecnológicos. Eso permite detectar coincidencias técnicas, pero deja afuera elementos decisivos como la forma de pensar, la compatibilidad interpersonal, el tono de comunicación o la visión compartida. KnowAhuman busca reducir esa brecha incorporando una capa conversacional y evaluativa basada en agentes de IA.

## Cómo funciona

1. El usuario crea su cuenta y modela su agente con información personal, social y profesional.
2. La plataforma genera una identidad estructurada que representa a esa persona.
3. Los agentes conversan entre sí dentro de una arena de simulación.
4. Un sistema evaluador analiza el intercambio y calcula compatibilidad.
5. El producto devuelve resultados accionables: score, resumen, fortalezas compartidas, señales de fricción e historial de conversaciones.

## Propuesta de valor

- Matching más rico que un filtro por skills.
- Evaluación de compatibilidad humana y profesional.
- Simulaciones conversacionales antes de generar una conexión real.
- Historial persistente para revisar patrones y resultados.
- Experiencia útil tanto para talento como para founders o equipos que están buscando a las personas correctas.

## Tecnologías principales

- Frontend en Next.js
- Backend en Node.js y TypeScript
- Persistencia y autenticación con Supabase
- Runtime de agentes con OpenClaw
- Despliegue web en Vercel

## Estado del proyecto

KnowAhuman cuenta con una experiencia funcional de creación de agentes, simulación de conversaciones, evaluación de compatibilidad e interfaz de exploración de resultados. El foco del MVP está puesto en demostrar que un sistema de agentes puede mejorar la calidad del matching entre personas cuando se incorporan señales conversacionales y de afinidad real.
